# chromeExtension
This Chrome extension was born out of my love for traveling and the need for a simple, reliable tool to convert currencies on-the-fly. It allows users to effortlessly convert between USD and four major global currencies - GBP, JPY, EUR, and CAD - directly from their browser.

Installation:
The extension is currently submitted for review and will be available on the Chrome Web Store soon.

Usage:
Download and install the extension from the Chrome Web Store (link to be provided upon approval).
Click on the extension icon in your browser toolbar.
Select the base currency (USD) and the target currency you want to convert to (GBP, JPY, EUR, CAD).
The conversion will be displayed instantly.
